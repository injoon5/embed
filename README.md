# YouTube Fullscreen Embed

A minimal page for displaying a YouTube video in fullscreen using the official embed format.
Vibe coded with ChatGPT. 


## Usage

with a video ID:
`https://e.ij5.dev/?v=dQw4w9WgXcQ`

With additional query parameters:
`https://e.ij5.dev/?v=dQw4w9WgXcQ?autoplay=1&rel=0`


## Notes

- uses `https://www.youtube.com/embed/VIDEO_ID`
- fullscreen layout
- minimal
